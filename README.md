# Venkat Sanjay - Developer Portfolio

A modern, responsive, and professional portfolio website built with pure HTML, CSS, and JavaScript. Features a dark/light theme toggle and is optimized for deployment on Vercel.

## Features

✨ **Modern Design** - Clean, professional UI with gradient accents  
🌙 **Dark/Light Theme** - Toggle between themes with persistent storage  
📱 **Fully Responsive** - Works seamlessly on all devices  
⚡ **Fast Performance** - Optimized for speed and accessibility  
🎨 **Smooth Animations** - Fade-in effects and hover interactions  
🔗 **Easy Navigation** - Smooth scrolling and sticky header  
🚀 **Ready for Vercel** - Zero configuration deployment  

## Sections

- **Hero** - Engaging introduction with call-to-action buttons
- **Projects** - Showcase of your best work with live demo links
- **Skills** - Technical expertise and tools you work with
- **Contact** - Get in touch form for inquiries
- **Social Links** - Connect on GitHub, LinkedIn, Twitter

## Customization

To customize this portfolio, edit the following in `index.html`:

1. **Name & Title** - Change "Venkat Sanjay" and subtitle text
2. **Navigation Links** - Update menu items and section IDs
3. **Project Cards** - Modify project descriptions and demo links
4. **Skills** - Add/remove skill items in the skills grid
5. **Social Links** - Update GitHub, LinkedIn, Twitter URLs
6. **Colors** - Modify CSS variables in `:root` and `[data-theme="dark"]`

## How to Deploy to Vercel

### Option 1: Using Git (Recommended)

1. **Initialize Git Repository**
   ```bash
   cd "c:\Users\venkat sanjay\portfoilo"
   git init
   git add .
   git commit -m "Initial portfolio commit"
   ```

2. **Push to GitHub**
   - Create a new repository on [GitHub](https://github.com/new)
   - Copy the repository URL
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
   git branch -M main
   git push -u origin main
   ```

3. **Deploy to Vercel**
   - Go to [Vercel Dashboard](https://vercel.com/dashboard)
   - Click "New Project"
   - Select your GitHub repository
   - Click "Deploy"
   - Your live URL will appear instantly!

### Option 2: Vercel CLI

1. **Install Vercel CLI**
   ```bash
   npm i -g vercel
   ```

2. **Deploy**
   ```bash
   vercel
   ```

3. **Follow the prompts** - Your site will be live with a `.vercel.app` domain

### Option 3: Drag & Drop

1. Go to [Vercel.com](https://vercel.com)
2. Drag and drop your portfolio folder
3. Your site deploys instantly!
